# What is this 

compare two numbers to get the lesser number, returns one of the values if the two numbers are equal 


# Installation 
`npm i is-less --save`

# Usage

```
import { isLess } from "is-less";

console.log(isLess(3, 5)); //=> 3
console.log(isLess(5, 2)); //=> 2
console.log(isLess(4, 4)); //=> 4

```
